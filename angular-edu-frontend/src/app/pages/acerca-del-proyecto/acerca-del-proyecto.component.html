Framework

¿Qué es Angular?

Framework de Google para construir SPAs robustas con TypeScript.

Angular es un framework de desarrollo web de código abierto diseñado para crear aplicaciones de una sola página. Utiliza TypeScript y una arquitectura basada en componentes para escalar con facilidad.

Características

Lo que hace único a Angular

Componentes

Encapsulan vista, lógica y estilos.

Data binding

Sincronizá datos de forma declarativa.

CLI potente

Generá código y automatizá builds.

Comandos

Angular en la terminal

ng new proyecto
ng serve -o
ng generate component feature
ng build --configuration production

Ventajas

Arquitectura clara, comunidad activa y soporte empresarial que acompaña cada release.
