Requisitos

Herramientas indispensables

Node.js

Instala la version LTS para acceder a npm y ejecutar Angular CLI.

npm

Gestor de paquetes para instalar dependencias y scripts.

VS Code

Editor recomendado con soporte para TypeScript y extensiones.

CLI

Instalacion global

Usa npm para instalar la herramienta oficial.

npm install -g @angular/cli
ng version

Primer proyecto

Crea y ejecuta en minutos

ng new mi-proyecto
cd mi-proyecto
ng serve -o

Conclusion

Con estos pasos ya contas con Angular CLI, un proyecto generado y el servidor en marcha sobre http://localhost:4200.
